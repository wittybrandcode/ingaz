## ما نريد الوصول إليه

نظام صلاحيات بسيط:

- **المدير** (is_manager) — الدور النظامي الوحيد، يملك كل الصلاحيات + رؤية كل شيء
- **الأدوار الديناميكية** — ينشئها المدير ويسميها ويختار صلاحياتها
- **الأعضاء** — لكل عضو دور واحد، يرون فقط ما كلفوا به

```
المدير → يكلف في المشاريع ← مكلفو المشروع (لديهم: projects.view, tasks.*, subtasks.*)
          └── يكلفون في المهام ← مكلفو المهمة
                  └── يكلفون في المهام الفرعية ← مكلفو المهمة الفرعية
                          └── يشاركون (submitted) + يعلقون + اختيار فائز
```
