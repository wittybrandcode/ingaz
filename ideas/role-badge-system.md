# نظام الشارات (Badge System) لقرص بروفايل الأعضاء

## الفكرة
نظام بصري من الشارات الصغيرة ترافق قرص (Avatar) بروفايل العضو، تظهر معلومات سريعة عنه مثل:
- تأهيله للتكليف
- رصيده (Credit Score)
- حالته (نشط / غير نشط)
- لاحقاً: أنواع تكليف أخرى

## التخطيط المرئي (من drawio)

```
        ╭───────────────╮
        │      ╭───╮    │
        │      │25 │    │  ← عداد (Credit Score / عدد الإنذارات)
        │      ╰───╯    │
        │  ╭───╮        │
        │  │   │   ╭───╮│
        │  │   │   │   ││  ← حالة (نشط/غير نشط)
        │  ╰───╯   ╰───╯│
        ╰───────────────╯
```

- القرص الكبير (240) = صورة/أفاتار العضو
- القص أحمر (100) = عداد (رصيد/إنذارات)
- القص أخضر (60) = مؤشر الحالة (نشط/غير نشط)
- القص أرجواني (80) = شارة الدور/التكليف

## قابلية التوسع
النظام مصمم بحيث نضيف أنواع شارات جديدة لاحقاً:
- `assign-qualified` → شارة تأهيل التكليف الأساسي
- `assign-project-leader` → شارة قائد مشروع
- `assign-task-manager` → شارة مدير مهام
- إلخ...

## المكونات
- `ProfileAvatar.tsx` — يلف Avatar العادي ويضيف الشارات حوله
- كل شارة لها: position (top-right, bottom-right, bottom-left, top-left), content, color, tooltip
